# Switch LLM

> A macOS application to run local models like Mistral, with the option to enhance conversations by
> switching to GPT-4.

<img src="./screenshot.png"/>

## Prerequisites

- macOS 14 (ARM)
- Node.js 20+
- Zig 0.12.0 (master)

## Getting started

Install npm dependencies:

```sh
npm install
```

Build the app:

```sh
npm run build
```

Start the app:

```sh
npm start
```
